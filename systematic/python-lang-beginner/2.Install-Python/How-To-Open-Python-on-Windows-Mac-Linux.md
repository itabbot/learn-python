## 4. 如何在 Linux 上打开 Python

输入 python3（或 python） 以启动 Python REPL：

```shell
# 启动REPL
python3
Python 3.10.12 (main, Nov 20 2023, 15:14:05) [GCC 11.4.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>>

# 关闭REPL
>>> exit()
```
